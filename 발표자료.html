Background

기획 배경

남성 그루밍 시장은 빠르게 성장하고 있지만, 정작 남성들은 "어디서부터 시작해야 할지" 모릅니다.

📈

커지는 시장, 낮은 접근성

남성 뷰티 수요는 매년 성장하지만, 대부분의 커머스·콘텐츠는 여전히 여성 사용자 중심으로 설계되어 있습니다.

🧴

높은 심리적 진입장벽

복잡한 성분 이름, 매장에서의 어색한 상담. 화장품이 처음인 남성에게는 시작 자체가 부담입니다.

💡

그래서 만들었습니다

"화장품, 처음이어도 괜찮아요." 버튼 몇 번이면 AI가 피부를 확인하고 관리 루틴까지 정해주는 서비스.

Problem

문제 정의

화장품 입문 남성이 겪는 4가지 장벽을 해결합니다.

🧭

시작점을 모른다

토너? 세럼? 로션? 무엇을 먼저 발라야 하는지, 순서조차 알기 어렵습니다.

📖

용어가 어렵다

히알루론산, 세라마이드… 성분 공부 없이는 제품 비교가 불가능합니다.

🏬

상담이 부담스럽다

매장 방문과 대면 상담이 어색해 구매를 미루거나 포기합니다.

❓

내 피부에 맞을까

광고·리뷰만으로는 내 피부(모공·유분·여드름·흉터)에 맞는지 확신할 수 없습니다.

Persona

페르소나

뷰티가 처음인 20~30대 남성 직장인 — "관리는 하고 싶은데, 뭘 사야 할지 모르겠어요."

🧑‍💼

김대리 (29세, 남성)

IT 회사 3년차 · 화장품 구매 경험 거의 없음

넓은 모공피지·유분화장품 입문
😰

고민

  • 면접·데이트 전 피부가 신경 쓰인다
  • 올리브영에 가도 뭘 골라야 할지 모른다
  • 지성 피부인데 로션이 맞는지 의문
🎯

니즈

  • 공부 없이, 버튼 선택만으로 진단
  • 순서대로 따라만 하면 되는 루틴
  • 같은 고민을 가진 사람들의 이야기

📊 데이터 기반 추천

올리브영 랭킹 데이터를 배치로 수집해 "카테고리 1위" 배지로 신뢰를 더했습니다.

🗣 사용자 언어로 설계

성분 용어 대신 "파인 흉터·넓은 모공"처럼 사용자가 실제로 쓰는 말로 선택지를 구성했습니다.

🛡 폴백 우선 설계

외부 데이터·API가 실패해도 시드 데이터로 항상 동작하도록 모든 경로에 폴백을 두었습니다.

Development

개발 내용 — 주요 기능

Streamlit 위에 단일 페이지 앱을 구현하고, Supabase·외부 API로 실시간성을 더했습니다.

🔬

AI 피부 분석

고민 선택 + 피부 설문 → 종합 스코어와 항목별 진단 리포트 제공. 공공 건강정보 데이터를 참고한 설명 문구.

🧴

5단계 제품 추천

스킨케어→선케어→베이스→아이→퍼퓸&바디 단계별 매칭. 올리브영 랭킹 1위 배지 자동 표기.

👔

TPO 스타일 가이드

출근·면접·데이트 등 상황 선택 시 실시간 날씨(Open-Meteo API) 기반으로 화장품+옷+헤어 조합 추천.

💬

커뮤니티

Supabase 연동 실시간 게시판. 탭 복귀 시 자동 새로고침으로 항상 최신 글 유지.

🎁

리워드 시스템

분석·결과 확인 등 미션 수행 시 포인트 적립, 제휴 서비스 연계 사용 시나리오 구현.

🔐

Google 로그인

OAuth 회원 인증으로 성별·연령대 기반 개인화("20대 남성 대비 비교") 제공.

Impact

기대효과

입문 장벽을 낮추면, 남성 뷰티의 첫 구매가 시작됩니다.

🚪

진입장벽 해소

30초 분석으로 "무엇부터"의 고민을 제거 — 뷰티 입문 남성의 첫 경험을 만듭니다.

🛒

구매 전환

진단 → 추천 → 제휴 링크로 이어지는 흐름이 탐색을 실제 구매로 연결합니다.

🔁

재방문 락인

커뮤니티와 리워드 포인트가 일회성 진단을 반복 방문 습관으로 바꿉니다.

📊

데이터 자산화

피부 고민·설문·구매 반응 데이터가 쌓여 추천 정확도와 상품 기획의 기반이 됩니다.

Hidden Mission

히든 미션 대응 방안

예상하지 못한 요구사항에도 코드 수정 없이, 또는 최소 수정으로 대응할 수 있게 설계했습니다.

🛡

폴백·시드 데이터 설계

올리브영 랭킹, 날씨 API, 이미지 파일 등 모든 외부 의존성에 폴백을 두어 어떤 환경에서도 데모가 멈추지 않습니다.

🔑

시크릿 분리 관리

Supabase 키·OAuth 자격 증명을 st.secrets로 분리해, 배포 환경이 바뀌어도 설정 파일만 교체하면 됩니다.

🖼

드롭인 리소스 교체

prod_*.png / look_*.jpg 파일만 폴더에 넣으면 코드 수정 없이 제품·스타일 이미지가 실사로 교체됩니다.

⚡

실시간 반영 구조

커뮤니티는 탭 포커스 복귀 시 자동 갱신 — 발표 중 즉석에서 글을 올려도 화면에 바로 반영됩니다.
